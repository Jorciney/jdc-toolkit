Privacy Policy for GitLab AI Code Reviewer

Last updated: 07/07/2025

Data Collection

This extension collects and processes the following data:

  • Claude API keys (stored locally in Chrome)
  • GitLab access tokens (stored locally in Chrome)
  • Code from GitLab merge requests (sent to Claude API for analysis)

Data Usage

Data is used solely for:

  • Authenticating with Claude and GitLab APIs
  • Analyzing code for review purposes
  • Displaying review results to users

Data Storage

API keys are stored locally in Chrome sync storage and never transmitted to third parties except the respective APIs (Anthropic Claude and GitLab).

Third-Party Services

This extension integrates with:

  • Anthropic Claude API for AI analysis
  • GitLab API for accessing repository data
